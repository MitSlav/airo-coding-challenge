## AIRO Coding Challenge

Installation:

- composer install
- php artisan key:generate
- php artisan jwt:secret - to Generate secret key for larawel JWT auth
- npm install
- npm run dev

For a development environment I use laravel herd. [php-open-source-saver/jwt-auth](https://github.com/PHP-Open-Source-Saver/jwt-auth) dependency is used for JWT management. For a frontend framework I chose vue.js because of familiarity.
